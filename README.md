##  Key Takeaways

- JavaScript has **8 data types**: `String`, `Number`, `BigInt`, `Boolean`, `Undefined`, `Null`, `Symbol`, and `Object`
- Use `typeof` operator to check the type of any variable
- `null` and `undefined` are different — `null` is intentional, `undefined` means not yet assigned
- Arrays and Objects are reference types, while the rest are **primitive types**
- **Type coercion** in JS can lead to unexpected results — always be mindful!
